# IM
一个C/S模式的聊天软件。</br>
实现功能主要有：</br>
	服务器端监控</br>
	注册帐号</br>
	添加、删除好友</br>
	文字、表情、截图聊天</br>
	查看好友资料</br>
	查看历史聊天记录</br>
	文件传输</br>
